Proyectos

AluraFlix - Challenge ONE

En este proyecto final se puso a prueba los conocimientos adquiridos en Alura Latam y Oracle Next Education. Se construyo un proyecto llamado AluraFlix, la idea del proyecto fue construir un sitio web en el cual se pudieran ver una lista de videos de Alura, clasificados en diversas categorías. La idea general de la interfaz, fue crear una similar a los sitios actuales de streaming, para que el usuario fácilmente pudiera navegar en ella según experiencias previas en otros sitios web.

En este proyecto se usó React JS para crear la interfaz de usuario, para manejar las rutas se usó React Router. Adicionalmente se usaron componentes de terceros como lo son: React Player y Embla carousel, sin embargo, estos componentes fueron adaptados según las necesidades del proyecto.

AluraGeek - Challenge ONE

Para este proyecto, se construyó una galería de productos dinámica, con la cual el usuario puede interactuar de tal forma que tiene la opción de: agregar, modificar y eliminar un producto. Adicionalmente en el detalle del producto puede calificarlo, esta calificación se ira promediando según el total de las calificaciones obtenidas.

En este proyecto se usó el DOM para la creación de elementos de forma dinámica. También se creó una API rudimentaria pero funcional, para manejar los datos del localStorage simulando una base de datos NoSQL, para de esta manera manejar más fácil la información de los productos guardados.
